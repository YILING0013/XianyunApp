# XianyunApp

基于NovelAI的图像生成软件

## 项目介绍
XianyunApp 是一个基于 NovelAI 技术的图像生成软件，用于生成高质量的图片。

## 主要技术
- **C#**: 主要编程语言，占比 95.5%
- **HTML**: 占比 3.6%
- **其他**: 占比 0.9%

## 安装与使用
### 环境要求
- .NET Framework
- 其他依赖库

### 安装步骤
1. 克隆仓库：`git clone https://github.com/YILING0013/XianyunApp.git`
2. 打开项目解决方案文件 `XianyunApp.sln`
3. 还原 NuGet 包：`nuget restore`
4. 编译项目：`dotnet build`

### 运行
运行生成的可执行文件，或者在 Visual Studio 中按 F5 进行调试。
